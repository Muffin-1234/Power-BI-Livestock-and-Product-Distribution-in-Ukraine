# Power-BI-Livestock-and-Product-Distribution-in-Ukraine
This project shows the distribution of livestock animals and products across farms in Ukraine

## Features
- Wool production by year
- Meat production by year
- Egg production by year
- Distribution of animal species across farms
- Distribution of animal products across farms

## Tools used
- Power BI Desktop
- DAX
